# Decision Copilot

**Live Demo:** https://jiandi-mu.github.io/decision-copilot-dem/

Decision Copilot is a lightweight AI decision assistant designed to help users turn vague needs into clearer choices.

## Core Idea
Users enter a scenario and a request, and the system returns a structured recommendation, including:
- best-fit direction
- key comparison dimensions
- risk reminders

## How It Works
1. The user selects a scenario.
2. The user enters a request or need.
3. The system generates a structured recommendation.
4. The result highlights direction, comparison dimensions, and risk reminders.

## Current Scenarios
- Electronics purchase
- Travel planning
- Restaurant selection
- Study tool selection

## Product Goal
Instead of only showing more information, Decision Copilot helps users move from information overload to faster decision-making.

This demo is currently being refined and updated. The repository contains the product concept, interaction design draft, and implementation progress.
